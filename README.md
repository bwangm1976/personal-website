# Ben Wang Personal Website

一个静态个人网站，包含：

- 自我介绍
- 个人履历
- 代表项目
- 技能标签
- 联系方式
- 维护页面
- 发表项目、论文、专利、个人荣誉
- 抖音号、学院主页等信息链接

直接打开 `index.html` 即可预览。

打开 `admin.html` 可以维护自我介绍、个人履历、代表项目、成果展示、信息链接和联系方式。

维护页会把内容保存在当前浏览器本地存储中。换电脑或换浏览器时，可以在维护页导出 JSON，再到新环境导入。

维护页包含简单登录验证和修改密码功能。账号与密码以哈希形式保存在网页端；这是静态网页的基础防护，不等同于服务器级权限控制。
